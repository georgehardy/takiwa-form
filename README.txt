Installation
Note
- Requires npm6
- Ensure you have the MongoDB service running. For MongoDB installation guide see [here](https://docs.mongodb.org/v3.0/installation/)

Instructions - Run the below in your terminal:

git clone https://gitlab.com/georgehardy/takiwa-form.git
cd takiwa-form
npm install
npm start
OR
npm test

Open http://localhost:8000/ in your browser to use the app!
